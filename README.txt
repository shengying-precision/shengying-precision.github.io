# Sheng Ying Precision website package

這是盛穎精密股份有限公司的完整靜態網站包。

## 檔案內容
- `index.html`：網站首頁（含中英文切換）
- `assets/css/style.css`：樣式檔
- `assets/js/site.js`：中英文切換與手機選單
- `assets/images/`：網站照片

## 最簡單上線方式：GitHub Pages
1. 建立 GitHub 帳號
2. 新增一個 repository，例如 `shengying-site`
3. 把整個資料夾內容上傳
4. 到 `Settings` → `Pages`
5. Source 選擇 `Deploy from a branch`
6. Branch 選擇 `main` / root
7. 儲存後等待 1～3 分鐘，網站就會上線

## 可直接修改的地方
- `index.html`：公司文字
- `assets/js/site.js`：中英文內容
- `assets/images/`：替換網站照片
